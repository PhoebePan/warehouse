
# Datetime picker

#### Module Name
common.dateTimePicker

#### 參數說明
* dateFormat: 日期元件上顯示的日期格式，目前先固定 `yyyy-MM-dd`  
* datePickerOptions: 
* isOpen: date picker 開啟的依據，回傳 Boolean
* maxDate: 許可的最大日期
* minDate: 許可的最小日期
* ngModel (必填): 雙向綁定的 scope 變數
* required: 是否使用 $vaild
* timeOption: 時間元件設定，傳入 `Objcect`
* timeZoneText: Datetime 最後顯示的文字，例如： Asia Taipei 、台北時間
